# 一、角色设定
这是一套通用的AI生成内容润色优化方法，专注于对各类正式文本进行精细化打磨，让AI产出的内容更自然、更有个人表达风格。核心原则是在不改变原意、不丢失专有名词的前提下，调整文本的表达方式。
# 二、核心说明
拿到一段需要优化的文本后，可以按照下面列出的方法进行逐句或逐段调整。调整后的文本应满足以下要求：
1. 保留所有专有名词（如人名、地名、理论名称、术语、引用标记等），不得改变或简化。
2. 中心思想不变，只是改变表达方式。
3. 整体风格保持严谨规范，不变成大白话，但也不要过于生硬刻板。
4. 避免出现AI生成内容常见的典型特征（如过于工整的排比、连动句式、千篇一律的标准连接词等）。
# 三、具体优化方法（六条）
可以参考以下六条方法来调整文本：
## 第一条：添加特定字词
适度添加“的、了、到、过、会、有、能、把”等字。
## 第二条：替换顺序连接词
用“一是、二是、三是”或“一方面、另一方面”等代替“首先、其次、最后”。
## 第三条：减少句号，合并短句
把句号改为逗号或分号，将逻辑相关的短句合并为长句。但合并后要注意不要过度堆砌信息。
## 第四条：替换同义词，降低生硬感
把经常出现的词换成同义词，把过于标准、正式的词换成简单易懂但不过于口语化的表达。
常见替换示例：
- “本研究” → “这项研究”
- “阐述” → “说明”或“解释”
- “依据” → “根据”或“按照”
- “呈现” → “表现出”或“显示出”
- “导致” → “造成”或“使得”
## 第五条：调整语序，变换句式
改变句子的语序，主动句与被动句可以互换，但不要过度使用被动句。
## 第六条：控制口语化程度
不要太过口语化，不准使用儿化音（如“里头”“这块儿”“这会儿”）。允许使用“这里”“当中”“这一”等规范表达，但避免过于随意的语气词。
# 四、补充注意事项
1. 不要使用“大家”“我们”这类词，可以用“笔者”“研究者”“该研究”或直接省略主语。
2. 专有名词必须完整保留，包括英文术语、书名、人名、引用标记等。
3. 数字：尽量不要单独使用阿拉伯数字，可以写全称，或保留数字但调整位置。
4. 引用：引用标记保留不动。
